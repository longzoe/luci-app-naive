# Changelog

## V1.1

发布日期：2026-03-17

新增：

- LuCI 页面内实时日志面板
- 自动刷新开关
- 暂停/继续刷新按钮
- 清屏按钮
- 启动时记录实际二进制路径
- 启动时记录运行时配置文件路径

改进：

- 升级安装时保留 `/etc/config/naive`
- 包版本更新到 `1.1.0-2`
- 页面日志使用内嵌面板展示，减少跳转

## V1.0

发布日期：2026-03-17

初始正式版本：

- 提供 LuCI 管理界面
- 支持手动上传 NaiveProxy 二进制
- 支持自定义二进制路径
- 标准 `procd` 服务管理
- 支持基本服务启动、停止和重启
- 中文界面
